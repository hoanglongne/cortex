# Template Brainstorm App Mới - CORTEX HUB

> Dùng template này khi brainstorm ý tưởng app mới với AI. Trả lời đầy đủ các câu hỏi để có thông tin cần thiết xây dựng trang chi tiết.

---

## 1. ĐỊNH NGHĨA CỐT LÕI

### 1.1 Tên App
**Câu hỏi:** App tên gì? (1 từ, ngắn gọn, gợi nhớ chức năng)
- Ví dụ: Solilo, Lexica, Dialecta, Oratio

**Trả lời:**
```
[TÊN APP]
```

---

### 1.2 Pure Function
**Câu hỏi:** App này giải quyết CHÍNH XÁC một vấn đề gì? (1 câu duy nhất, bắt đầu "Solves EXACTLY one problem:")
- ❌ SAI: "Giúp người dùng học từ vựng hiệu quả hơn"
- ✅ ĐÚNG: "Rewiring vocabulary recall through spaced-repetition stress tests"

**Hướng dẫn:**
- Phải cụ thể tới mức không thể hiểu sai
- Phải có động từ hành động (rewiring, eliminating, training, building, forcing)
- Phải nói rõ cơ chế (through X, via Y, under Z conditions)

**Trả lời:**
```
Solves EXACTLY one problem: [ĐỘNG TỪ HÀNH ĐỘNG] + [VẤN ĐỀ CỤ THỂ] + through/via [CƠ CHẾ CỤ THỂ]
```

---

### 1.3 Trụ Cột (Pillar)
**Câu hỏi:** App thuộc trụ cột nào trong 4 pillar?

**Chọn một:**
- [ ] **Biological** - Xử lý bộ nhớ cơ, phản xạ sinh học (ví dụ: Lexica, Echo, Visage)
- [ ] **Psychological** - Xử lý lo âu, tâm lý, cảm xúc (ví dụ: Solilo)
- [ ] **Cognitive** - Xử lý nhận thức, mẫu tư duy, phân tích (ví dụ: Dialecta, Syntaxis, Metamorph)
- [ ] **Combat** - Xử lý áp lực thời gian, tình huống thực chiến (ví dụ: Oratio)

**Giải thích lý do chọn:**
```
[GIẢI THÍCH]
```

## 2. CƠ CHẾ HOẠT ĐỘNG

### 2.1 Tagline
**Câu hỏi:** Câu tagline chính của app (2 câu ngắn, brutalist, thẳng thắn)

- Ví dụ: "Từ Vựng Không Để Biết. Để Nhớ Dưới Áp Lực."

**Trả lời:**

### 2.2 How It Works - 3 Bước
**Câu hỏi:** Quy trình sử dụng app gồm 3 bước nào? (Mỗi bước 1-2 câu ngắn)

**Bước 01:**
- Tiêu đề: `[ĐỘNG TỪ] + [ĐỐI TƯỢNG]` (VD: "Nghe Định Nghĩa")
- Mô tả: 
```
[MÔ TẢ BƯỚC 1 - 1-2 câu, tập trung vào hành động cụ thể người dùng làm]
```

**Bước 02:**
- Tiêu đề: `[ĐỘNG TỪ] + [ĐỐI TƯỢNG]`
- Mô tả:
```
[MÔ TẢ BƯỚC 2 - 1-2 câu, AI/hệ thống làm gì]
```

**Bước 03:**
- Tiêu đề: `[ĐỘNG TỪ] + [ĐỐI TƯỢNG]`
- Mô tả:
```
[MÔ TẢ BƯỚC 3 - 1-2 câu, kết quả/lặp lại như thế nào]
```

---

### 2.3 Đoạn Giải Thích Cơ Chế
**Câu hỏi:** Viết 2-3 câu giải thích TẠI SAO cơ chế này hiệu quả (dùng cho section "How It Works")

**Hướng dẫn:**
- Câu 1: Phủ nhận cách làm sai phổ biến
- Câu 2: Khẳng định cách làm đúng của app
- Câu 3 (optional): Kết quả cụ thể

**Trả lời:**
```
[2-3 CÂU GIẢI THÍCH]
```

## 4. TÍCH HỢP CORTEX

### 4.1 Tiêu Đề Section
**Câu hỏi:** Điểm yếu nào của user sẽ theo dõi qua apps? (Format: "[X] Của Bạn [Động từ]")
- Ví dụ: "Lo Âu Của Bạn Không Dừng Lại Ở Đây", "Từ Vựng Của Bạn Không Riêng Lẻ"

**Trả lời:**
```
[TIÊU ĐỀ CORTEX INTEGRATION]
```

---

### 4.2 Cách Tích Hợp Với Apps Khác
**Câu hỏi:** Dữ liệu từ app này sẽ ảnh hưởng đến app nào khác? Cách nào?

**Ví dụ Solilo:**
- → Oratio: Từ vấp trong Solilo → tranh luận trong Oratio
- → Dialecta: Giọng yếu → nghe giọng đó trong Dialecta

**Trả lời:**
```
App này → App X: [DỮ LIỆU THEO DÕI] → [ẢNH HƯỞNG CỤ THỂ]
App này → App Y: [DỮ LIỆU THEO DÕI] → [ẢNH HƯỞNG CỤ THỂ]
App này → App Z: [DỮ LIỆU THEO DÕI] → [ẢNH HƯỞNG CỤ THỆ]
```
